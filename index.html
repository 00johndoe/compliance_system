Measurement Methodology

Maturity Assessment Model

Each control is assessed on a 0-5 maturity scale (CMM-based), weighted by criticality, and aggregated across domains for composite scoring.

Level 0 — Non-Existent
Level 1 — Initial/Ad-hoc
Level 2 — Developing
Level 3 — Defined
Level 4 — Managed
Level 5 — Optimizing

Scoring Formula

Domain Score = (Σ (weight_i × maturity_i)) / (Σ (weight_i × 5)) × 100

Overall Score = Average of Domain Scores

Alignment = (Ghana Score + ISO Score) / 2

Key Insights & Observations

✓ Strong Convergence Areas

  • • Access control and identity management
  • • Incident response and management
  • • Network and endpoint security
  • • Vulnerability management
  • • Business continuity planning

⚠ Key Divergence Areas

  • • CII protection (Ghana-specific)
  • • Physical security granularity
  • • Personnel security depth
  • • National regulatory reporting
  • • Sector-specific coordination

📋 Recommendations for Ghana

  • • Expand physical security control coverage
  • • Add dedicated people security controls
  • • Include SDLC security requirements
  • • Add privacy-specific controls
  • • Include cloud security requirements

🌍 For Organizations in Ghana

  • • Implement both frameworks for comprehensive coverage
  • • Prioritize Ghana NCF for regulatory compliance
  • • Use ISO 27002 for detailed implementation guidance
  • • Ensure CII reporting obligations are met
  • • Align with Data Protection Act (Act 843)
